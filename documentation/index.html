HTML Structure

The template is based on pure HTML and CSS flex
The default css flex box system utilizes 12 columns, making for a 1170px wide depending on your viewport.
Below 480px viewports, the columns become fluid and stack vertically.
 
To add another row Create a div under row 2
  
<div class="cont-row">
        
        
            <div > { Row 1 } </div>

            <div > { Row 2 } </div>

</div>

CSS Structure

The organization of the CSS is one of our priorities.
These are the CSS file we're using in the template:
  
  - css/

JavaScript

jQuery - is a Javascript library that greatly reduces the amount of code that you must write.
For more information, please visit http://www.jquery.com/
 
All the JavaScript libraries and files are included at the bottom of every HTML page like this: 
<!-- Main JS -->
<script src=" ./javascript/file.js " type="text/javascript"></script>        
<!-- Jquery JS -->
<script src="./javascript/jquery.js" type="text/javascript"></script> 
<!-- Bootstrap JS -->
<script src="./javascript/bootstrspcdn.js" type="text/javascript"></script>

 

Fonts

To change the embedded font, please take a look in the head part of css/styles.css:
  
            @import url('https://fonts.googleapis.com/css2?family=Poppins:wght@400;700&display=swap');
            @import url('https://fonts.googleapis.com/css2?family=Pinyon+Script&family=Poppins:wght@400;700&display=swap');

 

To change the font, first go to http://www.google.com/webfonts choose a font and add the generated code to the top of global.css.
